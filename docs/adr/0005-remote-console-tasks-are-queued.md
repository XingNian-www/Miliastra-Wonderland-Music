# 远程指挥台任务异步入队

远程指挥台提交的播放控制、点歌、AI 点歌和控制台发言会进入主业务队列，HTTP 请求只返回“已入队”，不等待实际执行完成。这样可以避免 Web 请求绕过游戏输入串行化、歌曲锁、播放队列出队和控制台反馈规则，也避免在主业务队列被邀请、确认或好友发言阻塞时让远程页面长时间等待。远程任务使用和游戏内命令一致的执行、失败和重试语义，避免为远程入口维护第二套业务规则。

按需游戏截图是观察能力，不是远程任务；截图接口只抓取当前画面并返回压缩 JPEG，不自动激活窗口、返回一级或发送任何输入。

程序管理员启动是运行前置条件。远程指挥台不展示管理员状态、不尝试提权重启，也不提供窗口状态诊断；窗口激活只属于主业务流程执行前的准备动作。
